# Security Policy

## Supported Versions

現在、セキュリティサポートの対象は最新バージョンのみです。

| Version        | Supported |
| -------------- | :-------: |
| Latest         |     ✅     |
| Older versions |     ❌     |

## Reporting a Vulnerability

セキュリティ上の問題を発見した場合は、Issue で公開する前に、プロジェクト管理者へご連絡ください。

報告には、可能な範囲で次の情報を含めてください。

* 問題の概要
* 影響範囲
* 再現手順
* 想定されるリスク
* 回避策（存在する場合）

内容を確認後、対応方針を決定し、修正完了後に必要に応じて公開します。

## Disclosure Policy

利用者への影響を最小限に抑えるため、修正が完了するまでは脆弱性の詳細を公開しません。

修正後は、必要に応じて変更内容および影響範囲を公開します。

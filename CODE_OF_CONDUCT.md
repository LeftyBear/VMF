# Code of Conduct

## Our Pledge

本プロジェクトでは、すべての参加者が互いを尊重し、建設的かつ協力的なコミュニケーションを行うことを目指します。

年齢、経験、国籍、職業などにかかわらず、誰もが安心して参加できる開発環境を維持します。

---

## Expected Behavior

参加者には次の行動を期待します。

* 相手を尊重したコミュニケーションを行うこと
* 建設的な議論を行うこと
* 技術的根拠に基づいた提案を行うこと
* プロジェクトの設計方針を尊重すること
* レビューやフィードバックを前向きに受け入れること

---

## Unacceptable Behavior

次の行為は禁止します。

* 誹謗中傷、嫌がらせ、差別的な発言
* 個人攻撃や威圧的な言動
* スパムや無関係な投稿
* 他者の知的財産権を侵害する行為
* プロジェクト運営を妨げる行為

---

## Project Policy

本プロジェクトでは、以下の方針を遵守します。

* **VMF v1.0** の設計は凍結されています。
* 設計変更は直接行わず、改善案は **VMF v1.1 Candidate** として管理します。
* GitHub を唯一の正本（Single Source of Truth）として運用します。

---

## Enforcement

プロジェクト管理者は、本行動規範に反する行為に対して、必要に応じてIssue・Pull Request・コメントの編集、削除、または参加制限などの対応を行う場合があります。

---

## Scope

本行動規範は、本リポジトリにおけるすべての活動に適用されます。

* Issues
* Pull Requests
* Discussions
* Commit Messages
* Documentation
* Source Code
